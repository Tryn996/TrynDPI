# TrynDPI
Эта программа позволит вам обходить блокировки на территории РФ\
Оболочка сделана на Godot 4.6 а обход из bat файла\
Основанно на zapret
## Что дает вам программа?
Удобный интерфейс со множеством полезных функций, к которым относится:\
1)Удобное добавление сайтов которые вы хотите обойти\
2)Встроенный SpeedTest\
3)И удобные настройки приложения и обхода
## Как скачать?
1)Скачайте устоновщик и окройте его\
2)Пройдите устоновку и запустите\
3)Готово
